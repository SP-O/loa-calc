로아도쓰가 제공하는 것

로아도쓰(LoaDoss)는 로스트아크 유저를 위한 무료 계산 도구 모음입니다. 2026년 3월에 아비도스 제작 계산기로 시작해, 현재 다음 기능을 제공합니다.

  • 아비도스 제작 손익 계산 — 실시간 거래소 시세로 융화 재료 제작의 손익을 판단합니다. 가이드 보기
  • 재료 교환 계산 (기본 · 영지 농장) — 재료 교환의 골드 가치 손익을 계산합니다. 가이드 보기
  • 부족 재료 계산 — 목표 제작량 대비 부족한 재료와 구매 예산을 정리합니다.
  • 티파고 (티카투카 계산기) — 티카투카 미니게임의 최적 수를 추천합니다. 가이드 보기

모든 기능은 회원가입 없이 무료로 사용할 수 있습니다.

데이터 출처

시세 데이터는 스마일게이트가 공식 제공하는 로스트아크 오픈 API의 거래소 데이터를 사용합니다. 계산에 쓰이는 모든 재료의 현재 최저가와 최근 14일 평균·최고·최저 거래가를 주기적으로 불러오며, 재료 이름에 마우스를 올리면 해당 시세 정보를 차트로 확인할 수 있습니다.

시세 조회에는 캐시가 적용되어 있습니다. 접속자가 많아도 API 호출이 폭주하지 않도록 일정 주기로 갱신된 값을 함께 사용하는 구조이며, 표시되는 시세의 기준 시각은 화면에서 확인할 수 있습니다.

계산 방식의 투명성

계산 결과를 신뢰할 수 있으려면 "어떻게 계산했는지"가 보여야 한다고 생각합니다. 로아도쓰의 계산에는 다음이 반영됩니다.

  • 제작 수수료(골드)와 거래소 판매 수수료
  • 대성공 확률에 따른 기대 산출량
  • 직접 채집한 재료의 기회비용 (판매했을 때의 가치)
  • 교환 시 최소 거래 단위와 비율

주요 수치에는 툴팁으로 계산 근거를 함께 표시합니다. 계산이 실제와 다르다고 느껴지는 부분이 있다면 아래 피드백 경로로 알려주세요. 확인 후 수정하고 업데이트 내역에 기록합니다.

운영 원칙

로아도쓰는 개인이 운영하는 팬 사이트로, 스마일게이트 RPG 및 스마일게이트 스토브와 무관합니다. 게임 내 데이터의 권리는 원 저작권자에게 있습니다.

사이트 운영비(서버·도메인)는 광고 수익으로 충당하는 것을 목표로 하며, 광고가 계산기 사용을 방해하지 않도록 배치를 제한합니다. 개인정보 수집은 최소화하며, 자세한 내용은 개인정보 처리방침을 참고해 주세요. 테마(다크/라이트)와 즐겨찾기 같은 설정은 서버가 아닌 이용자의 브라우저에만 저장됩니다.

로아도쓰가 만들어진 이유

아비도스 제작은 "오늘 시세 기준으로 이득인가?"를 매번 따져야 하는 콘텐츠인데, 거래소를 오가며 재료 값을 하나씩 확인하고 수수료와 대성공까지 암산하는 과정이 생각보다 번거롭습니다. 처음에는 개인용 스프레드시트로 시작했던 계산을, 같은 고민을 하는 유저들이 쓸 수 있도록 웹 도구로 만든 것이 로아도쓰의 시작입니다.

그래서 로아도쓰의 개발 원칙은 "화면은 단순하게, 계산은 정확하게"입니다. 메인 화면은 계산기 하나에 집중하고, 배경 설명이나 소식 같은 읽을거리는 이 가이드 페이지들처럼 별도 공간에 두어 필요한 사람만 찾아볼 수 있게 했습니다.
