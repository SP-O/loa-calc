# 로아도쓰 (LoaDoss)

로스트아크 아비도스 제작 계산기. 실시간 경매장 시세를 연동해
제작 손익과 최적 재료 교환을 계산해 줍니다.

**[로아도쓰(LoaDoss)](https://loaclac-doss.vercel.app/)**

## 주요 기능
- **재료 교환 계산** — 보유 재료로 만들 수 있는 최대 제작 수량과
  최적 교환 경로 계산 (기본 / 영지 농장)
- **부족한 재료 계산** — 보유한 4T 생활 재료 수량에 맞춰 부족한 재료를 역산
- **제작 수익 계산** — 실시간 경매장 시세 기반 손익 분석
  (판매 수수료 5%·영지 제작 수수료·대성공 반영)
- **제작 랭킹 TOP3** — 생활별 제작 손익 순위 한눈에 비교
- **시세 차트** — 14일간 평균 거래가·거래량 그래프 제공
- **다크 / 라이트 모드** 지원

## 사용 기술
- Vue 3, Chart.js
- Vercel (서버리스 API) + 로스트아크 Open API
- Firebase, Matter.js (인터랙션 요소)

## 면책
본 사이트는 Smilegate RPG / Smilegate Stove와 무관하며,
로스트아크 Open API 데이터를 기반으로 합니다.
